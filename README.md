# SSV College - General Landing Page

## Overview
This is the main landing page for SSV College that showcases all departments including Law, MBA, AHS, CSE, ECE, Civil, Mechanical, and EEE.

## Features

### Hero Section
- **Design**: Matches the reference design with a teal left panel and decorative dots
- **Headline**: "Study anywhere, anytime with expert educators"
- **Search Bar**: Functional search with yellow button to filter courses
- **Hero Image**: Student with red book on yellow circular background
- **Floating Icons**: Animated decorative elements
- **Rating Badge**: 4.8 satisfaction rating with pulse animation

### Courses Section
All 8 departments are displayed with:
- Professional course images
- Department names
- Course counts
- Hover effects with elevation
- Click functionality (Law and MBA link to their respective pages)

### Statistics Section
- 10,000+ Students Enrolled
- 500+ Expert Educators
- 8 Departments
- 95% Success Rate
- Animated counters when section comes into view
- Glassmorphism design with teal gradient background

### Features Section
- Expert Faculty
- Flexible Learning
- Industry-Relevant Curriculum
- Certification Programs
- Icon-based cards with hover effects

### Call-to-Action Section
- Dark gradient background
- "Get Started Now" and "Explore Courses" buttons
- Smooth scroll to courses section

### Footer
- Quick Links
- Department Links
- Contact Information
- Responsive grid layout

## Technical Implementation

### Files
- `index.html` - Main HTML structure
- `styles.css` - Complete styling with modern design
- `script.js` - Interactive functionality

### Key Technologies
- **HTML5**: Semantic markup with proper SEO
- **CSS3**: Modern features including:
  - CSS Grid and Flexbox
  - CSS Variables for theming
  - Smooth transitions and animations
  - Glassmorphism effects
  - Responsive design
- **Vanilla JavaScript**: No dependencies, pure JS for:
  - Mobile navigation toggle
  - Smooth scrolling
  - Active navigation highlighting
  - Search functionality
  - Course card interactions
  - Scroll animations
  - Stats counter animation
  - Modal dialogs
